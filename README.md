# 🫁 Explainable Deep Learning Model for Early Pneumonia Detection Using Chest X-Ray Images

> **Final Year Project** — B.E. Computer Science & Engineering  
> Velammal Engineering College, Chennai

---

## 📋 Abstract

Pneumonia remains one of the leading causes of mortality worldwide, particularly among children and immunocompromised individuals. Early and accurate detection from chest radiographs is critical for timely clinical intervention. This project presents an **Explainable Deep Learning Model** that combines the diagnostic power of DenseNet-121 with three complementary Explainable AI (XAI) methods — Grad-CAM++, Integrated Gradients, and SHAP — to produce both highly accurate and interpretable pneumonia predictions from chest X-ray images.

A novel **XAI Consistency Score** is introduced as a cross-method spatial agreement metric, computed using Intersection over Union (IoU) and Pearson correlation across explanation maps. An **Ensemble Explanation Map** is further generated by weighted combination of all three methods, providing a unified consensus visualisation for clinical interpretability. The framework achieves an AUC-ROC of **96.90%** and a sensitivity of **99.49%** on the Kaggle Chest X-Ray dataset, demonstrating its potential as a reliable diagnostic aid.

## 🏗️ Model Architecture

### Backbone: DenseNet-121
- **Pre-training:** ImageNet
- **Fine-tuning dataset:** Kaggle Chest X-Ray (Pneumonia) — 5,232 images
- **Total parameters:** 7,610,498
- **Input resolution:** 224 × 224 (RGB, CLAHE-enhanced)

### Custom Classifier Head
```
DenseNet-121 Feature Extractor (frozen for 5 epochs)
    └── Linear(1024 → 512) → ReLU → Dropout(0.5)
    └── Linear(512 → 256)  → ReLU → Dropout(0.5)
    └── Linear(256 → 2)    → Softmax
```

### Training Strategy
- **Phase 1 (Epochs 1–5):** Backbone frozen, classifier head trained only
- **Phase 2 (Epochs 6–30):** Full network unfrozen, end-to-end fine-tuning
- **Optimiser:** AdamW | **Scheduler:** Cosine Annealing
- **Loss:** Cross-Entropy with class weighting (Normal: 1.9393, Pneumonia: 0.6737)
- **Early stopping patience:** 7 epochs

### Preprocessing
- CLAHE (Contrast Limited Adaptive Histogram Equalisation) for early pneumonia pattern enhancement
- Augmentation: ShiftScaleRotate, HorizontalFlip, Brightness/Contrast, GaussNoise (Albumentations)

## 🔬 Novel Contributions

### 1. XAI Consistency Score
A novel metric that quantifies spatial agreement across multiple explanation methods:
```
Consistency Score = mean(IoU(GradCAM, IG), IoU(GradCAM, SHAP), IoU(IG, SHAP))
                  + Pearson correlation coefficient
```
This metric provides a quantitative measure of explanation reliability, enabling clinicians to assess prediction trustworthiness beyond confidence scores alone.

### 2. Multi-XAI Ensemble Map
A weighted combination of all three explanation maps:
```
Ensemble = 0.4 × GradCAM++ + 0.3 × Integrated Gradients + 0.3 × SHAP
```
Produces a unified consensus heatmap that highlights regions consistently identified as diagnostically significant across methods.

### 3. Two-Phase Fine-Tuning
A structured transfer learning strategy specifically designed for medical imaging, preventing catastrophic forgetting while enabling domain-specific feature adaptation.

## ⚙️ Setup & Installation

### Prerequisites
- Python 3.12+
- NVIDIA GPU with CUDA 12.x (recommended: RTX 4070 or equivalent)
- Git

### 1. Clone the Repository
```bash
git clone <repository-url>
cd pneumonia_xai
```

### 2. Install Dependencies
```bash
python -m pip install -r requirements.txt
```

### 3. Download Dataset
Download the [Kaggle Chest X-Ray (Pneumonia)](https://www.kaggle.com/datasets/paultimothymooney/chest-xray-pneumonia) dataset and place it as follows:
```
data/chest_xray/
    ├── train/
    │   ├── NORMAL/
    │   └── PNEUMONIA/
    ├── val/
    │   ├── NORMAL/
    │   └── PNEUMONIA/
    └── test/
        ├── NORMAL/
        └── PNEUMONIA/
```

### 4. Train the Model
```bash
python src/train.py
```
The best model checkpoint will be saved to `outputs/checkpoints/best_model.pth`.
