# ensemble-explainability-framework-pneumonia-xai
Ensemble Explainability Framework with XAI Consistency Score for Pneumonia Diagnosis Using DenseNet-121

An explainable deep learning framework for early pneumonia detection from chest X-ray images. Integrates DenseNet-121 with Grad-CAM++, Integrated Gradients, and SHAP to produce a novel XAI Consistency Score and Ensemble Explanation Map. Achieves AUC-ROC of 96.90% and sensitivity of 99.49% on the Kaggle Chest X-Ray dataset. Built as a clinical-grade Streamlit web application.
